# HSM — Holy Shit Moment Generator

## Principles
1. Under 30 seconds from token to wow
2. Real results — hit live APIs, no demos
3. Personally relevant — use their profile data
4. Minimal input — one sentence
5. Visible change — verifiable in the real service

## By Platform

### Claude Code
Suggest based on profile. "Search CRM for VIP contacts" / "What's my Stripe balance?" / "Generate a blog post for {company}"

### Chrome Extension
Auto-scrape current LinkedIn profile → AI compose connection note → one-tap insert. 45 seconds install to LinkedIn message.

### Slack
/0n connect → bot pulls CRM dashboard → shows 5 recent contacts → offers to compose team update.

### WordPress
Token paste → shows N unpublished AI blog posts → one-click publish → live URL. "I published a blog without writing a word."

## Recovery
If first command fails: pivot to zero-config fallback (blog generation works for everyone).
